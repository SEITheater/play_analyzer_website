PlayAnalyzer.com is an open source project to provide insight and tools to theatrical artists. Currently we are in an open beta. To request access please send an email from a gmail account you wish your account to be linked to which lists your:

  • Name
  • Theater's Website (if applicable)
  • How you heard about PlayAnalyzer.com

We ask that if you utilize the analysis tools on this site during the beta that you actively provide feedback on the quality of the tools and suggest new tools that would be of use to our growing community. Additionally, if these tools provide novel insight into a production you are working on, we ask that you submit a write up with the content generated from this site to our gallery. This way all theatrical practitioners can benefit from this new contextual knowledge, even without generating their own visualizations.
